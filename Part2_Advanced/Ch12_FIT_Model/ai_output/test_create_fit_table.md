## 運費計算－FIT 測試表（含邊界值／正向／負向）

> 需求摘要
> - 活動期間
> - VIP 會員：消費 **超過 2000 元** → 免運
> - 一般會員：**一律收 100 元運費**

---

### FIT 測試表（Calculate Shipping Fee）

| 會員類型 | 訂單金額 | 預期運費 | 說明 |
|---------|----------|----------|------|
| VIP | 0 | 100 | 負向：金額過低，不符合免運條件 |
| VIP | 1999 | 100 | 邊界下界-1：未超過 2000 |
| VIP | 2000 | 100 | 邊界值：剛好 2000（不免運） |
| VIP | 2001 | 0 | 邊界上界+1：超過 2000，免運 |
| VIP | 5000 | 0 | 正向：高金額，免運 |
| 一般會員 | 0 | 100 | 正向：一般會員固定運費 |
| 一般會員 | 2000 | 100 | 邊界值：金額不影響結果 |
| 一般會員 | 2001 | 100 | 正向：一般會員一律收費 |
| 一般會員 | -1 | 錯誤 | 負向：不合法金額 |
| 未指定 | 3000 | 錯誤 | 負向：會員類型為必填 |
